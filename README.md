# DataMorph — PDF to Excel Converter

Smart PDF to Excel converter with OCR support for scanned / Type3-font PDFs.

## Features
- Upload any PDF and convert to Excel (.xlsx)
- OCR-powered extraction for scanned or custom-glyph PDFs
- Multi-strategy parsing for student attendance data
- Styled Excel output with auto-column widths
